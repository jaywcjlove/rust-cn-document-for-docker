Function core::arch::wasm64::v128_store1.54.0[−][src]

pub unsafe fn v128_store(m: *mut v128, a: v128)
This is supported on WebAssembly and target feature simd128 only.
Expand description

将 v128 vector 存储到给定的堆地址。

这个内部函数会发出一个对齐为 1 的 store。 虽然这是为了完整性而提供的，但并非绝对必要，您也可以直接存储指针：

ⓘ
let a: &mut v128 = ...;
unsafe { v128_store(a, value) };
// .. 是相同的 ..
*a = value;
Run

store 的对齐方式可以通过做一个没有这个内部函数的手动 store 来配置。

Unsafety

这个内部函数是不安全的，因为它需要一个裸指针作为参数，并且指针必须有效才能存储 16 个字节。

注意这个指针没有对齐要求，因为这个内部函数执行了一个 1-aligned store.
