Expand description

assert expression 产生 false 值 时将使用 Syscall。

Syscall 参数：

  • message: 指向应该输出的字符串的指针。
  • file: 指向与断言关联的文件名字符串的指针。
  • line: 与断言关联的行号。
  • function: 指向与断言关联的函数名称字符串的指针。

来源: PTX 互操作性。
