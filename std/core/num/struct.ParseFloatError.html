Struct core::num::ParseFloatError

1.0.0 · source · 
pub struct ParseFloatError { /* private fields */ }
Expand description

解析浮点数时可以返回的错误。

该错误用作 f32 和 f64 的 FromStr 实现的错误类型。

Example

use std::str::FromStr;

if let Err(e) = f64::from_str("a.12") {
    println!("Failed conversion to f64: {e}");
}
