Struct proc_macro::Punct

1.29.0 · source · 
pub struct Punct(_);
Expand description

Punct 是单个标点符号，例如 +、- 或 #。

像 += 这样的多字符运算符表示为 Punct 的两个实例，它们返回了不同形式的 Spacing。

impl Punct

source

pub fn new(ch: char, spacing: Spacing) -> Punct

根据给定的字符和间距创建一个新的 Punct。 ch 参数必须是语言允许的有效标点符号，否则函数将为 panic。

返回的 Punct 将具有默认范围 Span::call_site()，可以使用下面的 set_span 方法对其进行进一步配置。

source

pub fn as_char(&self) -> char

将此标点符号的值返回为 char。

source

pub fn spacing(&self) -> Spacing

返回此标点符号的间距，指示其是否紧随 token 流中的另一个 Punct，以便可以将它们组合为多字符运算符 (Joint)，也可以将其与其他 token 或空格 (Alone) 组合在一起，因此该运算符肯定具有结束了。

source

pub fn span(&self) -> Span

返回此标点符号字符的跨度。

配置该标点字符的跨度。
