Expand description

一个 trait，用于在 main 函数中实现任意的返回类型。

C 的 main 函数只支持将返回的整数作为返回的类型。 因此，实现 Termination trait 的每种类型都必须转换为整数。

默认实现将返回 libc::EXIT_SUCCESS 以指示成功执行。 如果发生故障，则返回 libc::EXIT_FAILURE。
