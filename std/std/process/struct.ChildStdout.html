Struct std::process::ChildStdout

1.0.0 · source · 
pub struct ChildStdout { /* private fields */ }
Expand description

子进程的标准输出 (stdout) 的句柄。

该结构体用于 Child 的 stdout 字段中。

当 ChildStdout 的实例被 丢弃 时，ChildStdout 的底层文件句柄将关闭。

impl From<ChildStdout> for Stdio

source§

fn from(child: ChildStdout) -> Stdio

将 ChildStdout 转换为 Stdio。

Examples

ChildStdout 将在引擎盖下使用 Stdio::from 转换为 Stdio。

use std::process::{Command, Stdio};

let hello = Command::new("echo")
    .arg("Hello, world!")
    .stdout(Stdio::piped())
    .spawn()
    .expect("failed echo command");

let reverse = Command::new("rev")
    .stdin(hello.stdout.unwrap())  // 在此处转换为 Stdio
    .output()
    .expect("failed reverse command");

assert_eq!(reverse.stdout, b"!dlrow ,olleH\n");
