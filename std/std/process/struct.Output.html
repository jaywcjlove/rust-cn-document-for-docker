Struct std::process::Output

pub struct Output {
    pub status: ExitStatus,
    pub stdout: Vec<u8>,
    pub stderr: Vec<u8>,
}
Expand description

完成的进程的输出。

Command 的 output 方法或 Child 进程的 wait_with_output 方法在结果中返回该值。

§stdout: Vec<u8>

进程写入 stdout 的数据。

§stderr: Vec<u8>

进程写入 stderr 的数据。
