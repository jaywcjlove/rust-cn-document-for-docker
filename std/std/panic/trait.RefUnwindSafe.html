Trait std::panic::RefUnwindSafe

1.9.0 · source · 
pub auto trait RefUnwindSafe { }
Expand description

一种标记 trait，表示共享引用被认为是 unwind 安全的类型。

即，该 trait 并非由 UnsafeCell 实现，UnsafeCell 是所有内部可变性的根。

这是 “辅助标记 trait”，用于为 UnwindSafe trait 提供 impl 块，有关更多信息，请参见该文档。
