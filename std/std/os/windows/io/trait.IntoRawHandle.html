一个 trait，表示消费一个对象并获得其原始 HANDLE 所有权的能力。

Required Methods§

source

fn into_raw_handle(self) -> RawHandle

消耗此对象，返回原始底层句柄。

此函数通常用于将底层句柄的所有权 ** 转移给调用者。 当以这种方式使用时，调用者是句柄的唯一所有者，并且一旦不再需要它就必须关闭它。

但是，转让所有权并不是严格要求的。 对严格转移所有权的 API 使用 Into<OwnedHandle>::into 实现。
