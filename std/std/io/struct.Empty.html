Struct std::io::Empty

1.0.0 · source · 
#[non_exhaustive]
pub struct Empty;
Expand description

始终处于 EOF 的 reader。

通常通过调用 empty() 来创建此结构体。 请参见 empty() 的文档以获取更多详细信息。
