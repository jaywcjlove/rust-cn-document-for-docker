Module std::collections::linked_list

1.0.0 · source ·
Expand description

具有所属节点的双向链表。

LinkedList 允许在恒定时间内在任一端推送和弹出元素。

NOTE: 使用 Vec 或 VecDeque 几乎总是更好，因为基于数组的容器通常更快，内存效率更高，并且可以更好地利用 CPU 缓存。
