Expand description

BTreeMap 上的游标。

Cursor 类似于迭代器，不同之处在于它可以自由地来回查找。

游标总是指向树中的一个元素，并以逻辑循环的方式进行索引。 为了适应这一点，有一个 “ghost” 非元素在树的最后一个元素和第一个元素之间产生 None。

Cursor 是使用 BTreeMap::lower_bound 和 BTreeMap::upper_bound 方法创建的。
