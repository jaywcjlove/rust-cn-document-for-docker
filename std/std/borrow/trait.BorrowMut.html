Expand description

用于可变借用数据的 trait。

作为 Borrow<T> 的同伴，这个 trait 允许一个类型通过提供一个可变引用作为底层类型借用。 请参见 Borrow<T>，以了解更多有关借用另一种类型的信息。

Required Methods§

source

fn borrow_mut(&mut self) -> &mut Borrowed

从拥有的值中借用。

Examples
use std::borrow::BorrowMut;

fn check<T: BorrowMut<[i32]>>(mut v: T) {
    assert_eq!(&mut [1, 2, 3], v.borrow_mut());
}

let v = vec![1, 2, 3];

check(v);
