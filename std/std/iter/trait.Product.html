pub trait Product<A = Self> {
    fn product<I>(iter: I) -> Self
    where
        I: Iterator<Item = A>;
}
Expand description

一个表示可以通过将迭代器的元素相乘来创建类型的 trait。

这个 trait 用于实现 Iterator::product()。 可以通过在迭代器上使用 product() 方法生成实现此 trait 的类型。

和 FromIterator 一样，这个 trait 应该很少被直接调用。
