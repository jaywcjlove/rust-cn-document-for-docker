Enum std::net::Shutdown

1.0.0 · source · 
pub enum Shutdown {
    Read,
    Write,
    Both,
}
Expand description

可以传递给 TcpStream::shutdown 方法的可能值。

Variants§

§

Read

TcpStream 的读取部分应关闭。

所有当前被阻止的和未来的 reads 将返回 Ok(0)。

§

Write

TcpStream 的写入部分应关闭。

所有当前被阻止和未来的 写入 将返回错误。

§

Both

TcpStream 的读取和写入部分均应关闭。

有关更多信息，请参见 Shutdown::Read 和 Shutdown::Write。
