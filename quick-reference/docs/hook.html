Hook 看起来是什么样的？

fn factorial(n) {
  if (n == 0)
    return 1;
  return n * factorial(n - 1);
}

Hook 具有类似于 C 的现代语法。

Hello, World

println("Hello, World!");
// Hello, World!

Hook 中的 Hello, World! 程序。

使用 Homebrew 安装

brew tap hook-lang/hook
brew install hook
hook --help

解释器可以通过 Homebrew 获取。

在 Windows 上安装

cd %tmp%
curl -sSLO https://raw.githubusercontent.com/hook-lang/hook/main/scripts/install.bat install

这是你在 Windows 上安装的方法。

布尔值

let x = true;
let y = false;

Bool 是布尔类型。所以，它可以是 true 或 false

数字

let x = 0;

数字可以是整数或浮点数

字符串

let empty = "";

字符串可以用单引号或双引号表示

区间

let range = 1..5;

println(range);
// 1..5

区间是整数的序列

数组

let fruits = ["apple", "banana", "cherry"];

println(fruits);
// ["apple", "banana", "cherry"]

数组是元素的序列

记录

println(p);

记录将字段映射到值

nil 值

let x = nil;
var y;

println(x); // nil
println(y); // nil

nil 表示没有值

假值

if (nil) "true" else "false";   // false
if (false) "true" else "false"; // false
if (true) "true" else "false";  // true
if (0) "true" else "false";     // true
if (1) "true" else "false";     // true
if ("") "true" else "false";    // true
if ([]) "true" else "false";    // true
if ({}) "true" else "false";    // true

只有 nil 和 false 是假值。

变量

变量

var x; // x contains nil
x = 5; // now, x contains a number
x = "foo"; // a string

println(x);

不可变变量

let x = 5;

x = 10; // error: cannot assign to immutable variable `x`

let y; // error: unexpected token `;`

不可变变量在声明时必须初始化。

作用域

let x = 5;
{
  let y = 15;
  println(x); // 10
  println(y); // 15
}
println(x); // 5
println(y); // error: variable `y` is used but not defined

当堆分配的变量超出作用域时，会自动释放。

遮蔽

let x = 5;

  println(x); // 10
}
println(x); // 5

变量可以被遮蔽。

循环

while 循环。

do while

do while 循环。

for

循环

loop {
  println("Press Ctrl+C to stop");

无条件的 loop 循环。

中断

var i = 0;

loop {
  if (i == 5) break;

  print(i);
  i += 1;
}
// 01234

使用 break 退出循环。

继续

var i = 0;

loop {
  i += 1;
  if (i % 2 == 0) continue;

  print(i);

if (i == 5) break;
}
// 135

使用 continue 跳过循环体的剩余部分。

函数声明

  return a + b;

let sum = |a, b| {

闭包

高阶函数

fn apply(f, x) {
  return f(x);
}

fn double(x) {
  return x * 2;
}

println(apply(double, 5));
// 10

函数可以作为参数传递或返回

函数的语法糖

fn factorial(n) =>
  if (n == 0) 1

当函数体是单个表达式时使用 =>

递归

fn fib(n) {
  if (n < 2)
    return n;
  return fib(n - 1) + fib(n - 2);
}

println(fib(10));
// 55

支持递归

错误处理

错误

//   at to_int() in <native>
//   at main() in example.hk:1

Hook 使用 panic 模式进行错误处理。当出现错误时，解释器会停止运行

语法错误

// syntax error: unexpected end of file
//   at main() in example.hk:1,25

Hook 具有严格的语法

崩溃

断言

返回错误

fn divide(a, b) {
  if (b == 0)
    return [nil, "division by zero"];
  return a / b;
}

if (let [ok, err] = divide(5, 0); ok) {
  println(ok);
} else {
  println(err);
}
// division by zero

使用一对来返回一个值和一个错误

传递错误

if (let [ok, err] = divide(5, 0); err) {
  return [nil, err];
}

传递错误而不处理它
