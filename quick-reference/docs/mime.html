MIME 类型 备忘清单

此备忘单列出了一些常见的 Web MIME 类型。 您可以查看包含所有已注册 MIME 类型的 IANA/MIME 媒体类型注册表。

入门

介绍

  • MIME 类型注册表关联特定的文件扩展名和文件名模式
  • MIME（多用途 Internet 邮件扩展）类型又名媒体类型
  • MIME 类型在 IETF 的 RFC 6838 中定义和标准化
  • 表示文档、文件或字节分类的性质和格式。
  • 对于 Internet 上的文件格式或格式内容
